Setup

To participate in a Data Carpentry workshop, you will need access to the software and data described below. In addition, you will need an up-to-date web browser.

  • Open Refine, version 3.3

    • {% comment %} Start of 'Setup Data' section {% endcomment %}

    Setup Data

    You need to download some files to follow this lesson:

    Download the following four files:

  • SAFI_clean.csv
  • SAFI_messy.xlsx
  • SAFI_dates.xlsx
  • SAFI_openrefine.csv

    • Place these 3 files in a folder you can easily find and access on your computer, titled datacarpentry-spreadsheets-openrefine or some other name that you will be able to remember is associated with this workshop.

    To learn more about the data, For more information about the dataset and to download it from Figshare, check out the Social Sciences workshop data page.
